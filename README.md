# 📈 Crude Oil Price Forecasting App

A machine learning-based desktop application for forecasting crude oil prices using XGBoost and time-series analysis.

---

## 🔧 Technologies Used
- Python
- Tkinter (GUI)
- XGBoost
- Pandas
- NumPy

---

## 📊 Features
- Predicts future crude oil prices
- Uses time-series forecasting
- User-friendly graphical interface
- Efficient data processing

---

## 🚀 How to Run
1. Install required libraries:
   pip install pandas numpy xgboost

2. Run the Python file or Jupyter Notebook

---

## 📌 Project Description
This project helps in analyzing historical crude oil prices and predicting future trends using machine learning techniques. It is designed as a desktop application for easy interaction.
